# **App Name**: Hackathon TeamUp

## Core Features:

- Profile Creation & Management: Users can create detailed profiles showcasing their skills, experience, and preferences for hackathon teams.
- Team Opening Posts: Users can post openings for their hackathon teams, specifying required roles, tech stack, and project ideas.
- Swipe-Based Teammate Discovery: A Tinder-style interface allows users to swipe through potential teammates based on AI-driven matching.
- AI-Powered Matchmaking: Utilize AI to analyze user profiles and team opening requirements to suggest the most compatible matches, ranking potential teammates and generating personalized summaries.
- Real-time Chat with Icebreaker Tool: Enable one-on-one conversations with matches, including AI-generated icebreaker tool suggestions to kickstart discussions.
- AI-driven Profile Verification: Implement a tool to detect fake or incomplete profiles using AI to analyze content, verify social links, and flag suspicious activity.
- Firestore Integration for Data Storage: Store user profiles, team openings, matches, and messages in Firestore for real-time data synchronization and scalability.

## Style Guidelines:

- Use Material Design 3 (Material You) dynamic color palettes derived from the user's wallpaper to create a personalized and adaptive color scheme.
- Prioritize accessible color contrast ratios to ensure readability and usability for all users.
- Employ the 'Roboto' font family as the primary typeface for its clear and versatile design, suitable for various text sizes and weights.
- Utilize Material Design 3's type scale for consistent and harmonious text styles across the app.
- Adopt Material Design Icons for a consistent and recognizable visual language throughout the interface.
- Use filled icons for primary actions and outlined icons for secondary or less frequent actions.
- Implement a bottom navigation bar for primary app sections (e.g., Home, Matches, Profile), ensuring easy access and discoverability.
- Use cards with rounded corners and subtle shadows to present user profiles and team openings in a visually appealing and organized manner.
- Incorporate subtle and purposeful animations for transitions, feedback, and visual cues to enhance the user experience.
- Use motion sparingly to avoid overwhelming users and maintain a clean and efficient interface.
